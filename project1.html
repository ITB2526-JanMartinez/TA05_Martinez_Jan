UPS Planning & Installation

A critical infrastructure project focused on power management and hardware protection within a corporate server room.

Project Description

This project involved the end-to-end management of a Uninterruptible Power Supply (UPS) installation for the company's main rack.


Key Responsibilities:

  • Load Analysis: Calculated the power consumption of all servers and switches to select the correct UPS capacity.
  • Rack Planning: Reorganized the rack layout to accommodate the heavy UPS units at the bottom for stability.
  • Installation: Physical mounting and cabling of the units.
  • Configuration: Set up network management cards for remote monitoring and configured graceful shutdown scripts for connected servers in case of power failure.
