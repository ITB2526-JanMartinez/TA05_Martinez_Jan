Secure Network Infrastructure

Designing a segmented network architecture to improve security posture using VLANs and Next-Generation Firewalls.

Project Description

In this academic project, I designed a secure network topology to mitigate internal threats and optimize traffic flow.


Technical Implementation:

  • Network Segmentation: Created distinct VLANs for Management, Employee Data, VoIP, and Guest Wi-Fi.
  • Firewall Configuration: Deployed a pfSense firewall with strict ACLs (Access Control Lists) to restrict inter-VLAN routing.
  • VPN Access: Configured OpenVPN for secure remote access for IT administrators.
  • Intrusion Detection: Enabled Snort IDS to monitor suspicious traffic patterns.
